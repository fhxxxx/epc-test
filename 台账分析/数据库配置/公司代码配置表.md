# 台账 Sheet 分析报告 - 公司代码配置表

## 1. 对应实体与表
- Sheet: `公司代码配置表`
- 实体: `CompanyCodeConfig`
- 表: `t_company_code_config`
- Mapper: `CompanyCodeConfigMapper`

## 2. Java字段清单（按实体）

### 2.1 实体自身字段
- `companyCode`：公司代码
- `companyName`：公司名称
- `financeBpAd`：财务BP AD账号
- `financeBpName`：财务BP姓名
- `financeBpEmail`：财务BP邮箱
- `isDeleted`：逻辑删除标记（0否/1是）

### 2.2 继承审计字段（AuditingEntity）
- `id`
- `createTime`
- `createBy`
- `createByName`
- `updateTime`
- `updateBy`
- `updateByName`

## 3. JSON结构化表达（jsonArray）

```json
[
  {
    "companyCode": "2320",
    "companyName": "上海睿景能源科技有限公司",
    "financeBpAd": "bp2320",
    "financeBpName": "张敏",
    "financeBpEmail": "bp2320@epc.local",
    "isDeleted": 0
  }
]
```

## 4. 说明
- 推荐把 `companyCode` 作为业务唯一键（逻辑删除条件下）。
- `companyCode`、`companyName` 为核心必填字段。
