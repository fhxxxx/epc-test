# 台账 Sheet 分析报告 - 税目配置表

## 1. 对应实体与表
- Sheet: `税目配置表`
- 实体: `TaxCategoryConfig`
- 表: `t_tax_category_config`
- Mapper: `TaxCategoryConfigMapper`

## 2. Java字段清单（按实体）

### 2.1 实体自身字段
- `seqNo`：序号
- `companyCode`：公司代码（可为空，表示通用）
- `taxType`：税种
- `taxCategory`：税目
- `taxBasis`：计税依据
- `collectionRatio`：征收比例
- `taxRate`：税率
- `accountSubject`：会计科目
- `isDeleted`：逻辑删除标记（0否/1是）

### 2.2 继承审计字段（AuditingEntity）
- `id`
- `createTime`
- `createBy`
- `createByName`
- `updateTime`
- `updateBy`
- `updateByName`

## 3. JSON结构化表达（jsonArray）

```json
[
  {
    "seqNo": "1",
    "companyCode": null,
    "taxType": "增值税",
    "taxCategory": "销项税额",
    "taxBasis": "不含税销售额",
    "collectionRatio": 100.00,
    "taxRate": 0.13,
    "accountSubject": "应交税费-应交增值税(销项税额)",
    "isDeleted": 0
  }
]
```

## 4. 说明
- `companyCode=null` 表示通用规则；有值表示公司覆盖规则。
- 金额与比率字段建议使用高精度数值类型（如 `BigDecimal`）。
