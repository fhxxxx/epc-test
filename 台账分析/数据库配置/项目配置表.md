# 台账 Sheet 分析报告 - 项目配置表

## 1. 对应实体与表
- Sheet: `项目配置表`
- 实体: `ProjectConfig`
- 表: `t_project_config`
- Mapper: `ProjectConfigMapper`

## 2. Java字段清单（按实体）

### 2.1 实体自身字段
- `companyCode`：公司代码
- `taxType`：税种
- `taxCategory`：税目
- `projectName`：项目名称
- `preferentialPeriod`：优惠期
- `isDeleted`：逻辑删除标记（0否/1是）

### 2.2 继承审计字段（AuditingEntity）
- `id`
- `createTime`
- `createBy`
- `createByName`
- `updateTime`
- `updateBy`
- `updateByName`

## 3. JSON结构化表达（jsonArray）

```json
[
  {
    "companyCode": "2320",
    "taxType": "企业所得税",
    "taxCategory": "企业所得税",
    "projectName": "鄂尔多斯（2023-2025三免期）",
    "preferentialPeriod": "减免期",
    "isDeleted": 0
  }
]
```

## 4. 说明
- `companyCode + projectName + taxType` 可作为业务去重组合（实现侧可再细化）。
- `preferentialPeriod` 建议统一枚举口径（如：减免期/无优惠期）。
