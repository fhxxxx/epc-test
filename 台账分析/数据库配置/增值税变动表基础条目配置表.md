# 台账 Sheet 分析报告 - 增值税变动表基础条目配置表

## 1. 对应实体与表
- Sheet: `增值税变动表基础条目配置表`
- 实体: `VatBasicItemConfig`
- 表: `t_vat_basic_item_config`
- Mapper: `VatBasicItemConfigMapper`

## 2. Java字段清单（按实体）

### 2.1 实体自身字段
- `itemSeq`：条目序号
- `companyCode`：公司代码（可为空，表示通用）
- `basicItem`：基础条目
- `isSplit`：是否拆分（Y/N）
- `isDisplay`：是否展示（Y/N）
- `isDeleted`：逻辑删除标记（0否/1是）

### 2.2 继承审计字段（AuditingEntity）
- `id`
- `createTime`
- `createBy`
- `createByName`
- `updateTime`
- `updateBy`
- `updateByName`

## 3. JSON结构化表达（jsonArray）

```json
[
  {
    "itemSeq": 1,
    "companyCode": null,
    "basicItem": "当月利润表主营业务收入",
    "isSplit": "Y",
    "isDisplay": "Y",
    "isDeleted": 0
  }
]
```

## 4. 说明
- `companyCode=null` 表示通用基础条目；有值表示公司级覆盖。
- `isSplit`、`isDisplay` 建议固定使用 `Y/N`。
